description CIM Database is the flexible transition model database. It acts as a common database between the Source System (Assetcenter, OVSD, ESL) and the Template Files from the Transition Model. The purpose is to be as close as possible to the Transition Model while allowing the flexibility to model the specifics of the Source system in the analysis phase.
