description This script will extract the Hardware Type information from ESL Relation Dump. The ESL Parent Relation Type has two types that can help to determine the Hardware Type (Box, Blade Enclosure, Blade Server). The Relation Type "Blade Server" is used in only 7 cases, investigation learned that it does not contain thrustworthy information. The Relation Type "Blade Enclosure" in a Parent Relation Type points from a Blade Server to a Parent System that is the Blade Enclosure.
